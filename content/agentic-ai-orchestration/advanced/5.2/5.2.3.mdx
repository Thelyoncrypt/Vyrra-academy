# LangGraph: State Machines, Checkpointing, and Human-in-the-Loop

> LangGraph extends LangChain with directed cyclic graphs for stateful multi-agent workflows.

## Lesson

LangGraph extends LangChain with directed cyclic graphs for stateful multi-agent workflows. It models agents as nodes, transitions as edges, and execution as a state machine.

Three Production Patterns:

Pattern 1: Human-in-the-Loop. 60% of production agent systems added human intervention points in 2026. LangGraph's interrupt API pauses at key decision points; after approval, execution resumes from the interruption point without restarting.

Pattern 2: Time-Travel Debugging. Checkpointers enable "forking" agent history. If an agent made a bad decision at Step 5, rewind to Step 4, change the prompt, and retry without losing prior progress.

Pattern 3: Long-Running Durable Execution. Agents can run for days. Persistence allows them to "sleep" and wake when external events (webhooks, approvals) trigger them.
