## 9.3 Kimi Agent Swarm: PARL Training

**Key Concepts:**

- **PARL (PArallel Reinforcement Learning)**: Multi-agent training framework

- **Agent Swarm**: Multiple agents collaborating on tasks

- **Specialized agents**: Different agents for different roles

- **Emergent coordination**: Agents learn to work together

- **Hierarchical organization**: Manager agents delegate to worker agents

### Agent Swarm Architecture

```text
┌─────────────────────────────────┐
│      Manager Agent               │
│  (Planning, delegation)          │
└──────────────┬──────────────────┘
               │

    ▼          ▼          ▼
```

## 9.4 Multi-Agent Orchestration

**Key Concepts:**

- **Orchestration**: Managing multiple agents working together

- **Communication patterns**: Direct, broadcast, hierarchical

- **Conflict resolution**: When agents disagree

- **Shared state**: Common memory and context

- **Error handling**: Recovery when agents fail

- **Scalability**: Adding more agents as needed

### Orchestration Patterns

| Pattern | Description | Use Case |

|---------|-------------|----------|

| Manager-Workers | Central manager delegates | Task distribution |

| Peer-to-Peer | Agents communicate directly | Collaborative problem-solving |

| Pipeline | Sequential agent chain | Data processing |

| Swarm | Emergent behavior | Exploration and search |

| Hierarchical | Multi-level management | Complex organizations |

## 9.5 Open-Weights & Self-Hosting

**Key Concepts:**

- **Open-weights models**: Model parameters available for download

- **Self-hosting**: Running models on your own infrastructure

- **Tools**: Ollama, LM Studio, vLLM, llama.cpp

- **Trade-offs**: Control vs. convenience, cost vs. quality

- **Hardware requirements**: GPU memory for large models

- **Quantization**: Reducing model size for efficiency

### Self-Hosting Options

| Tool | Best For | Hardware |

|------|----------|----------|

| Ollama | Easy local setup | Consumer GPU |

| LM Studio | GUI experimentation | Consumer GPU |

| vLLM | Production serving | Server GPU |

| llama.cpp | Edge devices | CPU/Minimal GPU |

| HuggingFace TGI | Cloud deployment | Cloud GPU |
