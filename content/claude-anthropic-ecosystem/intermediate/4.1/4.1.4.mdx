# Kimi Code: Installation, Commands, and MCP Compatibility

> Kimi Code CLI is distributed as a Python package via PyPI.

## Lesson

Kimi Code CLI is distributed as a Python package via PyPI. The recommended installation uses uv:

uv tool install --python 3.13 kimi-cli

Key commands: /plan toggles read-only plan mode; /yolo toggles auto-approve; /fork branches sessions for exploration; /compact summarizes conversation history; /mcp displays connected MCP servers. Kimi Code natively supports MCP, enabling integration with external tools including databases, documentation sources, and development tools. The CLI also supports the Agent Client Protocol (ACP) for IDE integration with VS Code, Zed, and JetBrains.
