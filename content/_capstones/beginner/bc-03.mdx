# Brand Identity Package

## Brief

Create a complete brand identity for a fictional small business using AI tools. The package must include a logo concept (AI-generated image), brand copy (tagline, about, product description), 3 social media assets, and a licensing compliance report.

## Requirements

- logo variations with documented prompts
- brand copy suite
- social media post images (1:1) with captions
- licensing compliance report identifying commercial rights per model
- cost breakdown
- prompt library with 5+ reusable templates.

## Deliverables

- logo variations with documented prompts
- brand copy suite
- social media post images (1:1) with captions
- licensing compliance report identifying commercial rights per model
- cost breakdown
- prompt library with 5+ reusable templates.
