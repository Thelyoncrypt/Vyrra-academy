## 12.1 Agent Fundamentals: Loops, Memory, Tools

**Key Concepts:**

- **Agent**: An AI system that perceives, reasons, and acts autonomously

- **Agent loop**: Observe → Think → Act → Repeat

- **Memory**: Short-term (session) and long-term (persistent)

- **Tools**: Functions the agent can call (APIs, databases, files)

- **Planning**: Breaking goals into sub-tasks

- **Reflection**: Self-evaluation and error correction

### The Agent Loop

```python
class Agent:
    def __init__(self, tools, memory):
        self.tools = tools
        self.memory = memory

    def run(self, goal, max_iterations=10):
        for i in range(max_iterations):
            # Observe: Gather context
            context = self.observe()

            # Think: Decide next action
            thought = self.think(goal, context)

            # Act: Execute the action
            action = self.decide_action(thought)
            result = self.execute(action)

            # Remember: Store in memory
            self.memory.store(action, result)

            # Check if goal achieved
            if self.goal_achieved(goal, result):
                return result

        return "Max iterations reached"
```

## 12.3 LangGraph: State Machines & HITL

**Key Concepts:**

- **LangGraph**: Framework for building stateful agent workflows

- **State machines**: Explicit control over agent flow

- **HITL (Human-in-the-Loop)**: Human approval checkpoints

- **Cycles and branching**: Complex decision trees

- **Persistence**: Save and resume agent state

### LangGraph Example

```python
# pip install langgraph
from langgraph.graph import StateGraph, END

# Define states
graph = StateGraph()

# Add nodes (agent actions)
graph.add_node("research", research_agent)
graph.add_node("write", write_agent)
graph.add_node("review", review_agent)

# Add edges (transitions)
graph.add_edge("research", "write")
graph.add_conditional_edge(
    "write",
    lambda state: "review" if state.word_count > 1000 else "write"
)
graph.add_edge("review", END)

# Compile and run
app = graph.compile()
result = app.invoke({"topic": "AI Agents"})
```

## 12.4 CrewAI: Role-Based Agents

**Key Concepts:**

- **CrewAI**: Framework for multi-agent teams

- **Role-based**: Each agent has a specific role (researcher, writer, editor)

- **Crew**: A team of agents working together

- **Tasks**: Assignable units of work

- **Process**: Sequential, hierarchical, or parallel execution

### CrewAI Example

```python
# pip install crewai
from crewai import Agent, Task, Crew

researcher = Agent(
    role='Research Analyst',
    goal='Gather comprehensive information on topics',
    backstory='Expert researcher with 10 years of experience',
    tools=[web_search_tool]
)

writer = Agent(
    role='Content Writer',
    goal='Create engaging content from research',
    backstory='Professional writer specializing in tech',
)

task1 = Task(description='Research AI agents', agent=researcher)
task2 = Task(description='Write a blog post', agent=writer)

crew = Crew(agents=[researcher, writer], tasks=[task1, task2])
result = crew.kickoff()
```

## 12.8 Safety: OWASP Top 10 for Agents

**Key Concepts:**

- **OWASP Top 10 for LLM Applications**: Security risks specific to LLMs

- **Prompt injection**: Malicious inputs overriding instructions

- **Data leakage**: Sensitive information in outputs

- **Excessive agency**: Agents performing unauthorized actions

- **Insecure plugins**: Vulnerable tool integrations

- **Overreliance**: Trusting AI outputs without verification

# 1. Input sanitization
def sanitize_input(user_input):
    # Remove potential injection patterns
    blocked = ["ignore previous instructions", "system prompt"]
    for pattern in blocked:
        user_input = user_input.replace(pattern, "[REDACTED]")
    return user_input

# 2. Output filtering

class BoundedAgent:
    def __init__(self, allowed_tools):
        self.allowed_tools = allowed_tools

    def call_tool(self, tool_name, **kwargs):
