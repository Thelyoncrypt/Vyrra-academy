# Structured Data: Schema.org and JSON-LD Implementation

> Structured data helps search engines understand page content.

## Lesson

Structured data helps search engines understand page content. JSON-LD (JavaScript Object Notation for Linked Data) is the recommended format. Common schemas include Article, Product, FAQPage, HowTo, Organization, and LocalBusiness.

{ "@context": "https://schema.org", "@type": "FAQPage", "mainEntity": [{ "@type": "Question", "name": "What is AI-assisted SEO?", "acceptedAnswer": { "@type": "Answer", "text": "AI-assisted SEO uses artificial intelligence tools to optimize..." } }]}
