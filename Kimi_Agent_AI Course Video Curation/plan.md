# AI Development Ecosystems Curriculum Blueprint — Execution Plan

## Overview
Design a complete, end-to-end learning programme covering modern AI development ecosystems, agentic workflows, automation systems, software engineering principles, design practices, and AI-powered product creation across Beginner, Intermediate, Advanced, and Expert levels.

## Stage 1: Deep Research (Parallel Swarm)
**Skill**: `deep-research-swarm` — Route A (Wide Search) for broad ecosystem coverage.

Deploy multiple research agents in parallel to gather current information on:

### Research Streams (all parallel):
1. **Claude Ecosystem Research** — Claude fundamentals, Claude Code, MCP, Artifacts, Projects, Tool Use, latest features, limitations, pricing, best practices
2. **OpenAI/Codex Ecosystem Research** — Codex CLI, ChatGPT for dev, OpenAI API, Assistants API, Function calling, Custom GPTs, o3 models, image/video generation, latest features
3. **Google AI Ecosystem Research** — Gemini models, Google AI Studio, Vertex AI, Google agents, Workspace AI, image/video generation tools, latest developments
4. **Kimi Ecosystem Research** — Kimi Agent, Kimi Swarm, Moonshot AI ecosystem, workflows, strengths/weaknesses, latest features
5. **Hermes Agent Research** — Identify what "Hermes Agent" refers to (possible: Nous Research Hermes models, Hermes API, or other), its architecture, use cases, setup, integration
6. **Image & Video Generation Models Research** — Seedance 2.0, GPT Image 2, Nano Banana Pro 2, Google imagen/video models, OpenAI Sora, Runway, Pika, Luma, Kling, Midjourney, Stable Diffusion, Adobe Firefly, Flux, etc.
7. **Agentic AI & Swarm Systems Research** — Multi-agent orchestration frameworks, goal-based systems, feedback loops, planning/reflection/evaluation loops, memory systems, popular frameworks (AutoGen, CrewAI, LangGraph, etc.)
8. **Native App Dev & Cross-Platform Research** — Swift/SwiftUI, Kotlin, React Native, Flutter/Dart, .NET MAUI, latest trends, AI integration
9. **Prompt Engineering & Custom Tools Research** — Latest prompt engineering techniques, MCP ecosystem, custom plugin development, tool-calling patterns

### Output: Validated research briefs saved to /mnt/agents/output/research/
