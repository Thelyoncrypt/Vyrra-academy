# Design Systems: Tokens, Components, and Documentation

> Design tokens are the single source of truth for color, typography, spacing, radii, and shadows - but they frequently go out of sync between design and code.

## Lesson

Design tokens are the single source of truth for color, typography, spacing, radii, and shadows - but they frequently go out of sync between design and code. The modern workflow uses Token Studio (Figma plugin) to manage tokens and export to multiple formats, Specify to collect tokens from Figma and push to GitHub repos, and the Figma API for custom automation scripts.

A design token file in JSON:

{ "color": { "primary": { "value": "#0066CC", "type": "color" }, "text": { "value": "#1A1A1A", "type": "color" } }, "spacing": { "sm": { "value": "8px", "type": "dimension" }, "md": { "value": "16px", "type": "dimension" } }}
