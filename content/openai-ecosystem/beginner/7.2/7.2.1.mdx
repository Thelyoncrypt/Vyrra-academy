# GPT-5 Family: GPT-5.2 through GPT-5.5, Capabilities, Pricing

> GPT-5.5 (released April 23, 2026) is the first fully retrained base model since GPT-4.5, representing a significant architectural departure.

## Lesson

GPT-5.5 (released April 23, 2026) is the first fully retrained base model since GPT-4.5, representing a significant architectural departure. Key technical shifts include a native omnimodal architecture where text, images, audio, and video flow through one unified model; hardware co-design around NVIDIA's GB200 and GB300 NVL72 rack systems; and a six-week release cadence establishing a new normal.

GPT-5.5 Variants and Pricing: - gpt-5.5: $5.00/1M input, $30.00/1M output; cached input $0.50/1M (90% savings) - gpt-5.5-pro: $30.00/1M input, $180.00/1M output; higher accuracy via parallel test-time compute - In Codex: 400K context window with Fast mode generating tokens 1.5x faster at 2.5x cost - In API: 1M context window - Pricing is 2x GPT-5.4's pricing ($2.50/$15), but OpenAI claims ~40% fewer output tokens per Codex task, making the effective cost increase ~20% for agentic workloads Benchmarks (GPT-5.5 vs GPT-5.4 vs Claude Opus 4.7):

GPT-5.4 (March 2026) offers five configurable reasoning levels (none, low, medium, high, xhigh), a 1M token context window, native Computer Use API scoring 75% on OSWorld-Verified (above 72.4% human baseline), and tool search reducing token usage by 47% on MCP Atlas benchmark.

GPT-5.2-Codex (January 2026) is purpose-built for software engineering: 80% SWE-Bench Verified, 87% CVE-Bench, with context compaction technology for sustained multi-file sessions and a 400K token context window. GPT-5.3-Codex (February 2026) is 25% faster, first to combine Codex and GPT-5 training stacks, and supports reasoning effort settings.

Safety Ratings (Preparedness Framework): Biological/Chemical: High; Cybersecurity: High, below Critical (increased from GPT-5.4); AI Self-Improvement: Below High.
