## 0.1 Welcome to AI Mastery 2026

Welcome to the most comprehensive, up-to-date AI course available in 2026. This course is designed for:

- **Complete beginners** who want to understand AI from first principles

- **Working professionals** who want to integrate AI into their daily workflow

- **Developers** who want to master AI coding agents and build agentic systems

- **Entrepreneurs** who want to build AI-powered products and automations

### How This Course Is Structured

The course follows a **dual-track approach** inspired by Anthropic Academy and OpenAI Academy:

```text
Track A: Non-Technical (Modules 0-4, 8, 11, 13)
  For professionals, students, educators, entrepreneurs

Track B: Developer Track (Modules 5-7, 9-10, 12, 14)
  For engineers, developers, technical builders

Track C: Everyone (All modules — pick and choose)
```

### Developer Tools Installation

```bash
# 1. Install Node.js (LTS) from https://nodejs.org

# 2. Install Claude Code
npm install -g @anthropic-ai/claude-code

# 3. Install OpenAI Codex CLI
npm install -g @openai/codex

# 4. Install Git
# macOS: brew install git
# Windows: https://git-scm.com/download/win
# Linux: sudo apt-get install git

# 5. Verify installations
claude --version
codex --version
git --version
```

### API Key Setup

```bash
# Claude Code - will prompt for auth on first run
claude

# Codex CLI - set OpenAI API key
export OPENAI_API_KEY="your-key-here"
# (Add to ~/.bashrc or ~/.zshrc for persistence)
```

## 0.3 Your Personalized Learning Path

### Path A: "I Want to Use AI Effectively" (Non-Technical)

Modules: 0 → 1 (basics) → 2 (ecosystem) → 3 (prompting) → 4 (Claude mastery) → 8 (Google AI) → 11 (image/video) → 13 (business)

### Path B: "I Want to Build with AI" (Developer)

Modules: 0 → 1 (foundations) → 2 (ecosystem) → 3 (prompting) → 5 (Claude Code) → 6 (MCP) → 7 (Codex) → 10 (comparison) → 12 (agents) → 14 (capstone)

### Path C: "I Want the Full Picture" (Complete)

All modules in order. Estimated: 120+ hours.
