# Building Your First Autonomous Agent

## Lesson

Exercise: Customer Support Agent with HITL

Build an autonomous support agent using either LangGraph or OpenAI SDK:

Triage agent classifies incoming requests (billing, technical, account)

Specialist agents handle each category with domain-specific tools

Human review gate pauses before any action costing >$50 or modifying account data

Risk tiering: Tier 0 (auto-execute reads), Tier 1 (notify), Tier 2 (synchronous approval), Tier 3 (multi-party approval). Audit logging: Every action logged with agent ID, reasoning trace, and tool invocation

Deliverable: Working agent with HITL, risk tiering, and full audit trails. Include documentation of architecture decisions and cost analysis.
