# Production RAG System with MCP Tools

## Brief

Production RAG System with MCP Tools.

## Requirements

- pgvector or Weaviate with BM25 + dense hybrid search
- cross-encoder re-ranker
- custom MCP server + community server
- citation system with confidence scores
- evaluation framework with automated metrics
- Docker containerisation with CI/CD
- prompt injection detection, input validation, rate limiting.

## Deliverables

- Deployed application with live URL, evaluation report, security checklist, cost analysis.
