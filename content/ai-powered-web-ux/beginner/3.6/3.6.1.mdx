# SEO Foundations

> Search Engine Optimisation (SEO) in 2026 requires understanding both traditional ranking factors and emerging AI-driven search dynamics.

- How Search Engines Work (2026)

## Lesson

Search Engine Optimisation (SEO) in 2026 requires understanding both traditional ranking factors and emerging AI-driven search dynamics. Generative Engine Optimisation (GEO) has emerged as a parallel discipline focused on making content citable and quotable for AI systems. This module covers technical foundations, on-page elements, semantic HTML, and Core Web Vitals.

Core Web Vitals at a Glance | Metric | Measures | Good Threshold | Mobile Pass Rate | |---|----|-----|------| | LCP | Loading performance | <= 2.5 seconds | 62% | | INP | Responsiveness | <= 200ms | 77% | | CLS | Visual stability | <= 0.1 | 81% |

Only ~48% of mobile websites pass all three Core Web Vitals simultaneously.

Structured Data Priority Types Pages with valid structured data (FAQ, HowTo, Article) appear 20-30% more often in AI-generated summaries than unstructured pages.

## Practice

### On-Page SEO Audit

On-Page SEO Audit (45 min): Analyse a webpage's title tag, meta description, H1 usage, heading hierarchy, URL structure, and internal linking. Create a prioritised improvement list.

### Semantic HTML Rewrite

Semantic HTML Rewrite (45 min): Rewrite a div-heavy page using semantic HTML. Add JSON-LD structured data for Article and FAQPage schemas. Validate with Google Rich Results Test.

### Core Web Vitals Measurement

Core Web Vitals Measurement (60 min): Use PageSpeed Insights and Lighthouse to measure a webpage's LCP, INP, and CLS. Identify the LCP element and propose optimisations (image format, preloading, CDN).

### GEO Content Optimisation

GEO Content Optimisation (30 min): Rewrite a blog paragraph for AI citation: add verifiable statistics, structure around clear claims, and include a FAQ section with schema markup.
