# AI Interactive Course Build — Execution Plan

## Context
User wants to build a new AI interactive course embedding their best YouTube videos + external high-quality content. Must filter out old/irrelevant content. Need to:
1. Catalog all videos from 3 playlists
2. Find Claude's official courses (clone structure)
3. Find Codex official courses (clone structure)
4. Create full interactive course: Neural Networks intro → Advanced

## Stage 1: Content Discovery (Parallel)
- Visit playlist 1: LLM Neural Network courses
- Visit playlist 2: Most recent videos
- Visit playlist 3: Additional videos (selective)
- Search YouTube for "Claude official course Anthropic"
- Search YouTube for "Codex course OpenAI"
- Search YouTube for "Claude AI tutorial 2025 2026"
- Search YouTube for "OpenAI Codex tutorial 2025 2026"
- Web search for Anthropic's official courses
- Web search for OpenAI's official Codex courses

## Stage 2: Content Analysis
- Extract video metadata from all playlists
- Identify outdated content (pre-2024, deprecated tools)
- Flag highest-quality recent content
- Map discovered videos to curriculum topics

## Stage 3: Course Architecture Design
- Build interactive course structure
- Map best videos to each module
- Identify gaps requiring new content
- Design interactive elements (quizzes, exercises, prompts)

## Stage 4: Output Production
- Full course blueprint document
- Video embed map with timestamps
- Interactive exercises per module
- Gaps analysis for new videos to record
