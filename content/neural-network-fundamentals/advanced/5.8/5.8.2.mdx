# Embeddings and Vector Search: RAG Implementation

> Retrieval-Augmented Generation (RAG) enhances LLMs by giving them access to external knowledge.

## Lesson

Retrieval-Augmented Generation (RAG) enhances LLMs by giving them access to external knowledge. The architecture: User Query -> Embed Query -> Vector DB Search -> Retrieve Top-K Chunks -> Build Context -> Append to Prompt -> Run LLM -> Generate Answer.
