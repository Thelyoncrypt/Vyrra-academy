## 8.1 Gemini Models: Pro, Flash, Flash-Lite

**Key Concepts:**

- **Gemini 2.5 Pro**: Most capable, best for complex reasoning and coding

- **Gemini 2.5 Flash**: Fast, cost-effective, good for most tasks

- **Gemini Flash-Lite**: Ultra-fast, cheapest option, good for simple tasks

- **Context window**: Up to 1M tokens (2M for select use cases)

- **Multimodal**: Text, image, audio, video input natively

- **Grounding**: Connect to Google Search for factual responses

### Model Selection Guide

| Task | Recommended Model | Why |

|------|-------------------|-----|

| Complex coding | 2.5 Pro | Best reasoning |

| Quick chat/QA | 2.5 Flash | Fast, capable |

| High-volume processing | Flash-Lite | Cheapest |

| Long document analysis | 2.5 Pro | 1M+ context |

| Image understanding | 2.5 Pro | Best vision |

| Real-time applications | Flash/Flash-Lite | Low latency |

### Gemini API Quickstart

```python
# pip install google-generativeai
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel('gemini-2.5-flash')

response = model.generate_content("Explain quantum computing in simple terms")
print(response.text)

# Multimodal: Image + Text
import PIL.Image
img = PIL.Image.open('photo.jpg')
response = model.generate_content(["Describe this image", img])
```
