# Google ADK: Enterprise Agent Development

> Google's ADK (Agent Development Kit) has been downloaded 7M+ times by Cloud Next 2026 and ships in Python, TypeScript, Go, and Java under Apache 2.0.

## Lesson

Google's ADK (Agent Development Kit) has been downloaded 7M+ times by Cloud Next 2026 and ships in Python, TypeScript, Go, and Java under Apache 2.0. Google reorganized its platform as the Gemini Enterprise Agent Platform, organized around four pillars: Build (Agent Studio for low-code, ADK for code-first), Scale (Agent Engine managed runtime, Memory Bank), Govern (Agent Identity, Agent Gateway, Model Armor), and Optimize (Agent Registry, monitoring, evaluation).

ADK provides Session, State, and Memory for context management; callbacks before/after model or tool execution for control and tracing; multi-agent primitives (sequential, parallel, loop, hierarchical); a built-in evaluation framework with evalsets and LLM-as-judge scoring; a local dev UI for testing; and one-command deploy to Vertex AI Agent Engine, Cloud Run, or GKE. A significant 2026 development: Google now hosts and manages MCP servers across all Google Cloud services, eliminating the need to build every MCP bridge from scratch.
