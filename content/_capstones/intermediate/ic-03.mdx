# AI-Powered SEO Content System

## Brief

Build a complete SEO content generation and optimisation system combining programmatic content creation with technical SEO fundamentals.

## Requirements

- Generate 10+ unique articles from structured data
- JSON-LD structured data for all content types
- technical SEO audit of a target website
- topic cluster map with internal linking
- Lighthouse scores >90
- content quality gates (readability, originality, fact-checking).

## Deliverables

- Generated content, SEO audit report, topic cluster visualisation, Lighthouse reports, content quality assessment.
